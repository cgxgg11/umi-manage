# umi manage

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

- 路由
- 菜单
- umi-request网络请求
- dva数据处理
- hooks使用
- antd
- TypeScript
- 多环境打包
